This repository is a compact reference for structuring Android UI tests with Espresso. It focuses on readable test flows, reusable actions, stable assertions, and a layout that can grow with a production test suite.

Use it as a starting point rather than a fixed framework. Selectors, synchronization, test data, and CI execution should be adapted to the behavior and risk profile of the application under test.
